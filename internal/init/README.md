# Setup Wizard

The guided `human init` setup that gets your project ready in minutes. It checks your tools, connects your trackers, and builds a secure container for AI agents to work in.

- Checks for required tools like Docker
- Connects trackers such as Jira and GitHub
- Generates your project config file
- Builds a secure devcontainer for agents
- Sets up a vault for your tokens
- Installs language servers for Claude Code
- Carries your Claude sessions into the container
