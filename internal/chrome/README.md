# Chrome Bridge

A tunnel that lets an AI agent running inside a container drive Chrome on your host machine, so the agent can browse, click, and screenshot real pages through your everyday browser.

- Connects containerized agents to host Chrome
- Tunnels browser traffic securely over the daemon
- Speaks Chrome's native messaging protocol
- Translates browser requests for the agent
- Carries screenshots and full page snapshots
- Requires a token before accepting connections
- Caps concurrent sessions to stay stable
