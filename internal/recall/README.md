# Cross-Tracker Search

`human` builds a local search index over the issues in all your trackers plus your Notion pages, so you can find any ticket from one command. Search runs against a fast local database with no network round-trip.

- Search every tracker and Notion at once
- Match against issue titles and full descriptions
- Relevance-ranked results with word stemming
- Filter results to a single tracker kind
- Search Notion pages and databases by content
- Refresh the index quickly with incremental updates
- Full re-sync option that prunes deleted tickets
- Output as plain text, a table, or JSON
