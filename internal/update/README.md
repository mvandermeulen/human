# Update Notifications

Keeps `human` aware of new releases and nudges you to upgrade, without ever slowing down a command. The check runs quietly in the background and tells you exactly how to update.

- Check for new releases in the background
- Cache results to avoid added latency
- Show an upgrade hint when one exists
- Suggest the right command for your install
- Skip notifications for dev builds
- Fail silently on network or cache errors
