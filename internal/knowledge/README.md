# Knowledge & Insights

`human` pulls product context from your docs, designs, and analytics so an AI agent can ground its work in what the team already knows. Today it connects to Notion, Figma, and Amplitude through one consistent set of commands.

- Search and read Notion pages as markdown
- Browse Notion database rows and properties
- Inspect Figma files, screens, and components
- Read design comments and export images
- Explore Amplitude events, funnels, and retention
- Look up individual users in Amplitude
- Reads tokens from config, env, or a vault
